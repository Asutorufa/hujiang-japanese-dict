运行:
```shell
npm install
# 日语
node hjjp.js
# 英语
node hjjpen.js
```
打包:
```shell
npm install
npm run pkg 
```

使用方法:  
```
# 日语
node hjjp.js 东京
# 英语
node hjjpen.js hello
```
